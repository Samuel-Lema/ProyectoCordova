# Cordova Escaner
App de Cordova con 2 botones, uno realiza una imagen y la muestra en la aplicación
y otro abre la camara con el escaner de barras/QR

# Referencias Plugins

1. Plugin de Camara
```
https://cordova.apache.org/docs/en/latest/reference/cordova-plugin-camera/
```

2. Plugin de Barcode Escaner
```
https://www.npmjs.com/package/cordova-plugin-barcodescanner
```
